SDG Choropleth Portfolio

This portfolio was created during a GIS course at Utrecht University. It contains a set of assignments that demonstrate spatial analysis, cartographic storytelling, remote sensing interpretation, and interactive web mapping using Leaflet.

The site includes the interactive SDG choropleth map, data processing and visualization outputs, historic map overlays, image classification results, and NDVI/deforestation comparisons.
